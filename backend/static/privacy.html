Politique de Confidentialité (RGPD)

1. Collecte des données

NutriAI collecte les données suivantes :

  • Informations de compte (email, nom d'affichage)
  • Descriptions de repas analysés
  • Données nutritionnelles calculées
  • Métadonnées d'utilisation (coûts IA, tokens utilisés)

2. Utilisation des données

Vos données sont utilisées pour :

  • Fournir le service d'analyse nutritionnelle
  • Améliorer la qualité du service
  • Calculer les coûts d'utilisation de l'IA
  • Respecter nos obligations légales

3. Base légale (RGPD)

Le traitement de vos données est basé sur :

  • Consentement : Vous acceptez cette politique en utilisant le service
  • Exécution d'un contrat : Nécessaire pour fournir le service
  • Intérêt légitime : Amélioration du service et prévention de la fraude

4. Partage des données

Vos données peuvent être partagées avec :

  • OpenRouter : Pour l'analyse IA (conformément à leur politique de confidentialité)
  • Hébergeur : Render/Railway pour le stockage sécurisé
  • Autorités : Si requis par la loi

Nous ne vendons jamais vos données à des tiers.

5. Vos droits (RGPD)

Conformément au RGPD, vous avez le droit de :

  • Accès : Consulter vos données personnelles
  • Rectification : Corriger vos données
  • Suppression : Demander la suppression de vos données
  • Portabilité : Récupérer vos données dans un format structuré
  • Opposition : Vous opposer au traitement de vos données
  • Limitation : Limiter le traitement de vos données

Pour exercer vos droits, contactez-nous à : contact@nutriai.app

6. Conservation des données

Vos données sont conservées :

  • Pendant la durée d'utilisation du service
  • 3 ans après la fermeture du compte (obligations légales)
  • Les données peuvent être anonymisées pour des statistiques agrégées

7. Sécurité

Nous mettons en œuvre des mesures techniques et organisationnelles appropriées pour protéger vos données :

  • Chiffrement des données en transit (HTTPS)
  • Accès sécurisé aux bases de données
  • Authentification forte pour les comptes administrateurs
  • Surveillance des accès

8. Conformité AI Act

NutriAI utilise l'IA de manière transparente :

  • Modèle utilisé : GPT-3.5 Turbo via OpenRouter
  • Les utilisateurs sont informés de l'utilisation de l'IA
  • Les résultats sont présentés comme des suggestions, pas des avis médicaux
  • Disclaimers médicaux présents dans l'application

9. Modifications

Cette politique peut être modifiée. Les utilisateurs seront notifiés des changements importants.
